# 鱼缸日记

鱼缸日记是完全开源的养鱼记录APP，所有的数据均保存在本地。

这个仓库是作者学习vibe coding的即兴作品，也希望能帮助到每一个养鱼佬。

## 功能

| 功能             | 数据实现 | 表单UI | 说明       |
| :--------------- | :------: | :----: | :--------- |
| 添加鱼缸         |    ✅    |   ✅   | 主键，自增 |
| 添加生物         |    ✅    |   ✅   | 主键，自增 |
| 添加生物状态变化 |    ✅    |   ✅   | 主键，自增 |
| 添加设备         |    ✅    |   ✅   | 主键，自增 |
| 添加耗材         |    ✅    |   ✅   | 主键，自增 |
| 添加维护记录     |    ✅    |   ✅   | 主键，自增 |
| 添加周期计划     |    ✅    |   ✅   | 主键，自增 |
