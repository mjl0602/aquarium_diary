# 鱼缸日记

鱼缸日记是完全开源的养鱼记录APP，所有的数据均保存在本地。

这个仓库是作者学习vibe coding的即兴作品，也希望能帮助到每一个养鱼佬。

## VibeCoding开发

本项目尽量使用VibeCoding的形式进行

1. 主页: 使用`Deepseek V3.2`生成，然后由人类作者古法编程优化
2. 数据类: 使用`Kimi 2.5快速`生成
3. 表单: 使用`Deepseek V3.2`生成

## 操作路径

### 添加鱼缸

1. 填写鱼缸名称，鱼缸类型
2. 完善剩余信息

页面需要针对常见操作优化:

- 开缸
- 换水
- 检疫
- 换棉
- 购买/出售生物
- 下药
- 检测水质
